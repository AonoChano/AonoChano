我是一名華中師範大學電子資訊科學與技術專業的本科生。我熱衷於構建事物、理解其運作原理，並透過實踐學習。以下是我感興趣的領域：

- 🤖 AI代理和LLM應用
- 🛠 開發者工具和自動化
- 💻 軟體工程和開源項目
- 🔌 嵌入式系統和電子
- 📚 在機器學習和自動控制方面打下更堅實的基礎

我喜歡嘗試新想法、閱讀原始碼，並構建一些小專案來解決日常開發中的實際問題。
