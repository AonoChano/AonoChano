私は華中師範大学の電子情報科学技術専攻の学部生です。ものづくり、仕組みの理解、実践による学習に情熱を注いでいます。以下は私が興味を持っている分野です：

- 🤖 AIエージェントとLLMアプリケーション
- 🛠 開発者ツールと自動化
- 💻 ソフトウェアエンジニアリングとオープンソースプロジェクト
- 🔌 組み込みシステムとエレクトロニクス
- 📚 機械学習と自動制御のより強固な基礎を築くこと

私は新しいアイデアを試したり、ソースコードを読んだり、日常開発での実際の問題を解決する小さなプロジェクトを構築することを楽しんでいます。
