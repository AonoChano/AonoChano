## About Me

Hi, I’m **Satoru**.

My name is written as **暁**.  
Yes, it is easy to read it as *Akira*, but in my case it is **Satoru**. Rare does not mean impossible.

I’m currently a student at **Central China Normal University / CCNU**, majoring in **Electronic Information Science and Technology**.

I like things that feel precise, expressive, and a little romantic: clean systems, low-level logic, visual interfaces, signal processing, embedded ideas, retro anime aesthetics, and tools that make people feel more capable.
